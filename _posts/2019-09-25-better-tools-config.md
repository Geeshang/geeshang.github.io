# 工具配置

## 统一快捷键(shortcut)

- 统一高频使用的快捷键
  我常用编辑器是 VS Code，常用的 IDE 有 PyCharm 和 IntelliJ IDEA，还有一个快速开发环境是 Jupyter Notebook。它们的默认快捷键是不一致的，统一各种开发工具的快捷键有助于提高开发效率。

- 删除不需要的快捷键

1. reformat

   command + option + f

2. show function signature（IDEA: quick documentory）

   command + option + h

3. rename

   command + option + n
